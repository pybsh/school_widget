### Introduction
An app that allows you to check Korean elementary, middle, and high school timetables and meal menus as widgets. <br/>
This app was created to help adapt to timetables at the beginning of the semester. <br/>
It was developed using Flutter to be available on all platforms.

### Key Features
- Fetch information using NEIS API
- Support widgets tailored to each OS and display information
- Configure school, grade, and class
